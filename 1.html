    Hypertext Markup Language(HTML) is the standard markup language for creating web pages and web applications. html Document symbols are also available for HTML, allowing you to quickly navigate to DOM nodes by id and class name. You can also work with embedded CSS and JavaScript.

    However, note that script and style includes from other files are not followed, the language support only looks at the content of the HTML file.
